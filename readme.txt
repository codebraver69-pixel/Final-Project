Project description:
-This website helps people see the different conditions of each country and gain more info about it.Also you're able to challenge yourself in flag knowledge.

technologies used:
-6 HTML pages
-CSS
-Javascript
-Bootstrap
-frankfurter Api
-mymemory Api
-restcountries
-localstorage

How to use the website:
-first you open the website and you are greeted with an introduction, you can click on any of the divs presented to go to another page or you can simply use the navigation bar to go to another page
-Second you go to the currencies page where you are able to compare between 2 currencies so you can know the value
-third you go to the translating page where you are able to translate anything between 4 languages
-fourth you go the comparison page where you can compare the capital,name,flag,population,timezone... of the 2 countries
-during all these pages you are able to add items to your favorites which will be presented in the favorites page
-finally you are able to play a small flag quiz game and gain more knowledge about each flag

Main Features:
-As said earlier i used an api so that you can compare,translate,and test your knowledge
-I used the local storage for the darkmode button so your preference will remain as you'd like
-I used another local storage for the favorites so they'd all be saed and able to view whenever you like
